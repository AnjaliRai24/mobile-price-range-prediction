# mobile-price-range-prediction
A simple Streamlit web app that predicts the price range of mobile phones using a LightGBM machine learning model. Users can input phone specifications like RAM, battery power, and screen resolution, and the app classifies the device as Low, Mid, High, or Flagship price range instantly.
